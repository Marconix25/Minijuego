# Minijuego
Mini proyecto (evidencia tecnica)
Este proyecto lo realice en un proyecto escolar, es un "videojuego"; pero para mi es mas que nada un minijuego. Tiene menu de inicio, y al darle start inicia el juego, aparecen enemigos y entre mas pasa el tiempo mas dificil se hace, esta principalmente inspirado en TLOZ (The legend of zelda; Minish cap"), ya que use los sprites del juego para poder usarlos dentro del mio. He realizado mas proyectos pero en conjunto, asi que como tal este seria uno donde trabaje individualmente,usando Python y trabajando con las librerias PY.game.
